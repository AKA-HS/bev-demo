# bev-demo
BEV(Bird's Eyes), an autonomous driving mode with an aerial view perspective was adopted.
Abstract: 
This project demo is mainly about AI pipeline of collecting truth-data from crowded-sources mapping, such as road test vehicle, mass production vehicle.
Aim: 
We perform industrial-grade cleaning and compliant de-identification of multimodal data such as videos, images, point clouds, and GPS data in ground truth datasets, providing the most intelligent data source for VLM models and their teams to conduct simulation training and scene mining.
Demo:
Some demo porjects about Bev, mainly about AI-Involved back-end software.
